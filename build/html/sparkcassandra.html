Setting up cassandra¶

The current build is Cassandra 3.9, on top of Java 8 (JDK 1.8)

java version "1.8.0_111"
Java(TM) SE Runtime Environment (build 1.8.0_111-b14)
Java HotSpot(TM) 64-Bit Server VM (build 25.111-b14, mixed mode)

CQLSH version 5.0.1

To set up cassandra, follow the same instruction as you can find on the internet (and cross fingers, hopefully you won’t get into any compatibility and versioning troubles) :

  1. Go to the apache project page, download and untar the ball file.
  2. Add the CASSANDRA_HOME to the path.
  3. Edit cassandra.yaml and put the right IP adresse(s) (‘127.0.0.1’ for localhost).
  4. Launch Cassandra with the command ‘cassandra’ (Be sure that JAVA_JOME is setted)
